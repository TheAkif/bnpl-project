# BNPL Frontend

Minimal React application using Vite.

## Prerequisites

- **Node.js** v18.x or v20.x (LTS)  
- **npm** v8 or higher  

> Other Node versions may produce compatibility warnings with Vite.

## What is Vite?

[Vite](https://vitejs.dev/) is a lightning-fast build tool and development server for modern web applications, offering instant hot‑module replacement and optimized production builds.

## Quick Start

1. **Install dependencies**  
   ```bash
   npm install
   ```

2. **Start the dev server**  
   ```bash
   npm run dev
   ```  
   Visit `http://localhost:5173`

3. **Build for production**  
   ```bash
   npm run build
   ```

4. **Preview the production build**  
   ```bash
   npm run preview
   ```
